● The game ●

The numerical version of the game is usually played with 4 digits. Here there will be 4 digits.
You will have to guess the secret code with only 7 turns.
If the number you entered exists and correctly placed, a point will be written under the Bull's column.
If the number exists but not correctly placed, a point will be written under the Cow's column.
So now good luck and have fun !
